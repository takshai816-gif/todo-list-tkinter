# 📝 To-Do List Application & Reminders

A simple desktop To-Do List app built with Python's Tkinter GUI library, developed as a micro project for I Year BTech (AI & Data Science).

## Features
- Add new tasks
- Delete selected tasks
- Clear all tasks (with confirmation)
- Mark tasks as done by double-clicking
- Live task counter (total / pending)

## Concepts Used
- Tkinter GUI programming
- Listbox widget
- Buttons and event binding
- Python Lists (task storage)
- State management (done/pending status)

## How to Run
1. Make sure Python 3 is installed (Tkinter comes built-in, no extra installs needed).
2. Clone this repository or download `todo_app.py`.
3. Run the app:
```bash
   python todo_app.py
```
